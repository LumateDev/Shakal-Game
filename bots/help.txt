1. Создать файл с расширением .pl
2. Написать код в файл и сохранить
3. На сайте (https://www.swi-prolog.org/Download.html) скачать SWI-Prolog под свою операционную систему
4. Установить SWI-Prolog
5. Запуск бота:
    5.1 Открыть терминал
    5.2 Перейти в директорию с ботом
    5.3 Запустить SWI-Prolog командой swipl
    5.4 Загрузить файл с ботом командой [bot_filename].
    5.5 Для запуска бота ввести команду start_bot. (название основной функции из файла с ботом)
